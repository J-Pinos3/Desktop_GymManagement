# Gym Desktop Application With C++ And Qt

### The technical features of this product are:
- MVC Architecture.
- MySQL Database.
- Gmail SMTP Server.
- C++ 11 Std.
- OpenSSL Integration For Security.
- Qt Widgets.

### In this app the user can:
- Create An Account Rather As Admin, Trainer Or Staff. 
- Make An Appointment With A Physiotherapist.
- Make An Appointment With A Nutritionist.
- Check Gym Assystance.
- Check If A Customer's Plan Has Expired.
- Generate Invoices.
- Send Emails With Invoices.
- Update Customer's Profile.
